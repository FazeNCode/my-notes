<h3> NETWORK </h3>

List detailed information about the network
```
ipconfig /all
```


Test network connection
```
tnc 10.0.0.160
```

To list the protocols
```
netstat -antp
```

Address resolution protocol 
```
arp -a
```


Print the routing table.
```
route print
```

To list the users on the machine
```
net user
```

Type in  [show helper]
```
netsh
```
